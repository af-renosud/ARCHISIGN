Step 1: Create server/middleware/asyncHandler.ts

Implement a standard Express async wrapper function (e.g., const asyncHandler = (fn) => (req, res, next) => Promise.resolve(fn(req, res, next)).catch(next)). This will catch any unhandled promise rejections and forward them to the global error handler.

Step 2: Create server/middleware/validators.ts
